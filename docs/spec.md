# 概要

 100人以上が参加する屋外撮影会があり、出演者と屋外エリアとタイムテーブルをうまく可視化したい。
 HTMLとJSでアプリを作ります。

# 仕様

 - A, B, C, D ブロックに出演者はわかれており、それぞれ色分けされています
 - 時間は1部, 2部, 3部にわかれています
 - 屋内エリアと屋外エリアのふたつの画像があるのでそこに名前をプロットします
 - 名前はブロック色の角丸長方形の座布団に濃いめの色
 - 画面下部には部ごとのボタンがあり、押すと出演者がエリアにアニメーションで移動します
 - 画面下部に出演者を選択する「出演者選択」ボタンがあり、ダイアログが開いてお気に入りをチェックする事ができます。検索もできます。永続します。
 
 # 進め方
 
 - 出演者の画像からテーブルを生成します。それぞれ1枚ずつ確認してファイルを生成して。
